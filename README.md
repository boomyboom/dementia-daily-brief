# 치매·AD 데일리 브리프

치매·알츠하이머병 연구, 진단 소프트웨어 규제·인허가(신의료기술 포함), 시장·투자,
경쟁사 동향을 매일 오전 9시(KST)에 자동 수집·요약하는 정적 대시보드.

- `index.html` — 대시보드 (GitHub Pages로 서빙)
- `briefs/YYYY-MM-DD.json` — 일별 브리핑 데이터
- `briefs/manifest.json` — 날짜 목록
- `BRIEFING_GUIDE.md` — 브리핑 생성 에이전트 작성 규칙

갱신은 Claude Code 스케줄 에이전트가 `BRIEFING_GUIDE.md`를 따라 수행한다.
